ptrdiff
=======

[![Hackage](https://img.shields.io/hackage/v/ptrdiff.svg)](https://hackage.haskell.org/package/ptrdiff)

Type "safe" pointer differences. Mostly makes it easier to get things right when working with `plusPtr`.
